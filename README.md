
# Vividly

This is video chatting web application
(Still under development)

## Features

- Video Calling
- Messaging 
    - Text Messages
    - Video Messages
    - Audio Messages
    - Stickers and GIFs  
- Messaging during Video Call
- Screen Sharing During video call
- An AI that `automatically adjusts the lighting` according to your surroudings.
